## Before
- Old site: slow, unoptimized, stock photos, no schema, no mobile optimization
- No analytics
- No structured data
- English-only (Greek market)

## After
- Single-page, 10-section site
- Mobile responsive + dark/light mode
- EL/EN bilingual
- Lighthouse: Perf 85-90, A11y 96, SEO 100
- Schema: LocalBusiness + FAQPage + areaServed
- Analytics-ready (GA4 placeholder)
- WebP images throughout
- sitemap.xml + canonical + OG

## Key Decisions
- Single service area section instead of 20 location landing pages
- Highlighted trust word ("Εγγύηση") in green
- Real hero photo (client workspace), no stock
- Logo variants: dark/white for light/dark mode
- Compact header with flag-only language toggle
- Trust metrics in unified glass bar

## Lessons Learned
1. Client intake is the biggest bottleneck — lack of assets delayed production
2. GBP optimization > website SEO for local service businesses
3. Real photos > stock photos (client was initially hesitant, result justified it)
4. Compact header with smaller nav text prevents wrapping on Greek words
5. Service area FAQ question adds natural local SEO without landing pages
