# Project Scoring Framework — AION Web Solutions

## Purpose
Not every client is worth taking. This framework helps you evaluate opportunities objectively before committing time to a proposal.

## Scorecard

| Criterion | Weight | Scale | Score |
|---|---|---|---|
| **Budget** | 25% | 1-10 | /10 |
| **Material Readiness** | 20% | 1-10 | /10 |
| **Upsell Potential** | 15% | 1-10 | /10 |
| **Project Complexity** | 15% | 1-10 | /10 |
| **Portfolio Value** | 15% | 1-10 | /10 |
| **Timeline Fit** | 10% | 1-10 | /10 |
| **Total** | 100% | | **/10** |

### Budget (25%)
| Score | Meaning |
|---|---|
| 1-3 | < €500 |
| 4-5 | €500-1,000 |
| 6-7 | €1,000-2,000 |
| 8-9 | €2,000-4,000 |
| 10 | €4,000+ |

### Material Readiness (20%)
Do they have what we need to start?
| Score | Meaning |
|---|---|
| 1-3 | Nothing — no logo, photos, content |
| 4-5 | Logo exists, nothing else |
| 6-7 | Logo + some photos |
| 8-9 | Most assets ready |
| 10 | Everything delivered on Day 1 |

### Upsell Potential (15%)
Can this client grow?
| Score | Meaning |
|---|---|
| 1-3 | One-off project, no recurring need |
| 4-5 | Possible maintenance |
| 6-7 | Likely maintenance + occasional updates |
| 8-9 | Recurring retainer potential |
| 10 | Long-term partner potential |

### Project Complexity (15%)
How complex is the build?
| Score | Meaning |
|---|---|
| 1-3 | Custom development, integrations, multi-page |
| 4-5 | Heavy customization |
| 6-7 | Some customization |
| 8-9 | Straightforward single-page |
| 10 | Perfect fit for existing template |

### Portfolio Value (15%)
Does this project improve your portfolio?
| Score | Meaning |
|---|---|
| 1-3 | No portfolio value |
| 4-5 | Low value |
| 6-7 | Good addition |
| 8-9 | Strong portfolio piece |
| 10 | Dream project for your niche |

### Timeline Fit (10%)
Does the timeline work for you?
| Score | Meaning |
|---|---|
| 1-3 | Urgent / impossible timeline |
| 4-5 | Tight but possible |
| 6-7 | Comfortable |
| 8-9 | Plenty of time |
| 10 | Flexible |

## Decision Rules

| Total Score | Action |
|---|---|
| ≥ 8 | ✅ Proceed with proposal |
| 6 - 7.9 | ⚠️ Proceed with caution — identify risks first |
| < 6 | ❌ Decline or refer elsewhere |

## Red Flags (Automatic Decline)
- Budget < €500 for a full website
- No assets and unwilling to provide them
- "My nephew can do it for €200"
- Multiple decision-makers with conflicting opinions
- Unrealistic timeline
- Unclear scope ("just make it look nice")
- Client has a history of unpaid invoices (check)

## Notes
- Score BEFORE writing a proposal
- Re-score if new information emerges during discovery
- Keep a log of scored projects → helps calibrate over time
