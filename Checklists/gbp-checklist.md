# GBP (Google Business Profile) Optimization Checklist — Service Business v1

## Why GBP Matters
For local service businesses, GBP drives more leads than website SEO in most cases.
Google prioritizes proximity + GBP authority for "near me" and "[service] [location]" queries.

## Profile Setup
- [ ] Business name matches website exactly
- [ ] Address matches website + schema (NAP consistency)
- [ ] Phone matches website (local number preferred)
- [ ] Website URL points to correct page
- [ ] Business category is as specific as possible (e.g., "Computer Repair Shop" not just "Service")
- [ ] Secondary categories selected (max 10 relevant)
- [ ] Service area defined (city-level, not radius if you travel)
- [ ] Hours match website + schema

## Photos
- [ ] Logo uploaded (matches website logo)
- [ ] Cover photo (hero-quality, workspace or team)
- [ ] Interior photos (3-10, real workspace)
- [ ] Team photos (if applicable)
- [ ] Service photos (examples of work)
- [ ] Photos match style/quality of website images
- [ ] New photos added monthly (Google favors active profiles)
- [ ] Photo geotag corrected (if possible)

## Reviews
- [ ] Minimum 10 reviews before ads / heavy marketing
- [ ] Reply to EVERY review (positive and negative) within 24-48h
- [ ] In replies, naturally mention services without keyword stuffing
- [ ] Encourage reviews from different service areas
- [ ] Create a review link shortcut: `https://g.page/r/[YOUR_ID]/review`
- [ ] Add review link to website header or footer
- [ ] Never buy fake reviews

## Posts
- [ ] Weekly post: offer, tip, or update
- [ ] Use photos in posts (2x engagement)
- [ ] Posts include CTA button
- [ ] Promote seasonal offers (e.g., back-to-school laptop check)

## Q&A
- [ ] Pre-populate 5-10 Q&A pairs (FAQ content)
- [ ] Pin the most important question
- [ ] Monitor for new questions and answer same day

## Attributes
- [ ] "Identifies as" attributes (if applicable)
- [ ] Service options: "On-site services" / "Online estimates"
- [ ] Payment methods accepted
- [ ] Languages spoken

## Ongoing
- [ ] Check GBP Insights weekly (search queries, calls, direction requests)
- [ ] Respond to new reviews within 48h
- [ ] Post at least 1x/week
- [ ] Add new photos monthly
- [ ] Update hours for holidays
- [ ] Monitor for duplicate listings (common issue)
