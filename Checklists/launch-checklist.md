# Launch Checklist — Service Business v1

## Pre-Launch
- [ ] All sections complete and content-final
- [ ] Mobile responsive — test all breakpoints
- [ ] Dark/light mode — toggle + visual check
- [ ] Language switch — EL/EN both verified
- [ ] All links functional (nav anchors, tel:, mailto:, external)
- [ ] Form submission works (or placeholder if backend pending)
- [ ] Images load correctly (no broken paths)
- [ ] Lighthouse targets met (>90 Perf, >95 A11y, >95 SEO)
- [ ] Google Analytics 4 installed + verified
- [ ] Google Search Console property verified + sitemap submitted
- [ ] Microsoft Clarity installed + recording
- [ ] sitemap.xml exists + accurate
- [ ] robots.txt allows indexing + points to sitemap
- [ ] canonical tag points to production URL
- [ ] favicon + apple-touch-icon added
- [ ] OG preview valid (Facebook Sharing Debugger)
- [ ] No console errors
- [ ] Custom domain configured (if applicable)
- [ ] SSL active
- [ ] 301 redirects from old domain (if migrating)

## Post-Launch (Week 1)
- [ ] Monitor GA4 for traffic sources
- [ ] Monitor Clarity for user behavior (click maps, scroll depth)
- [ ] Check Search Console for crawl errors
- [ ] Request 3-5 Google Reviews from first clients
- [ ] Verify mobile-first indexing in Search Console

## Post-Launch (Month 1)
- [ ] Review GA4 data: top pages, CTAs clicked, phone calls
- [ ] Review Clarity recordings for UX issues
- [ ] Update Search Console with any new pages
- [ ] Collect client feedback for iteration
