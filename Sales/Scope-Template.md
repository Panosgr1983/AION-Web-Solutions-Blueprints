# Scope Template

## Included (Always)
- 1-page website with baseline sections (Hero, Why Choose Us, Reviews, Services, Process, FAQ, Service Area, Contact, Footer)
- Mobile responsive across all breakpoints
- Dark mode + light mode
- Greek + English language
- SEO: LocalBusiness Schema, FAQPage Schema, canonical, OG, sitemap, robots.txt
- Analytics: GA4, Search Console, Clarity setup
- Image optimization: WebP conversion, lazy loading, dimensions
- Performance: Lighthouse 90+ target
- Favicon + apple-touch-icon
- 1 round of revisions

## Not Included (Unless Explicitly Agreed)
- Content creation (copywriting)
- Professional photography
- Logo design
- Custom illustrations
- E-commerce functionality
- Booking / appointment system
- Blog / CMS
- Custom integrations (CRM, ERP)
- Social media management
- Paid ads (Google/Facebook)
- Domain registration or hosting
- Email setup
- Ongoing maintenance (separate retainer)

## Client Responsibilities
- Provide all content (text, photos, logo) within 1 week of start
- Respond to revision requests within 48h (otherwise project proceeds)
- Provide GBP access if optimization is included
- Provide hosting access if migration is needed
- Review and approve final version within 1 week of delivery

## Revision Policy
- 1 round included (up to 10 change requests)
- Additional rounds: €150/round
- Changes requiring new sections: quoted separately
- Scope creep: any request outside original agreement will be quoted before work begins
