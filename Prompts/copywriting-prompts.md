# Prompts — Service Business v1

## Hero Copy
```
Write a hero section for a local [service business type] in [location].
Format: H1 that states "Problem + Solution with Guarantee".
Subtitle: short, benefit-focused, 15-25 words.
CTA1: "Call Now" (action-oriented).
CTA2: "Free Diagnosis" (risk-reversal).
Tone: professional, confident, helpful.
```

## FAQ Generation
```
Write 9 FAQ questions and answers for a local [service business] website.
- Questions as the customer would search/ask
- Answers 1-3 sentences, direct, no fluff
- One question must address service area / range
- Include typical concerns: pricing, timing, quality, guarantees
Business: [brief description]
Location: [city/area]
```

## Service Area Copy
```
Write a natural "Service Area" section for a local business website.
Format: "We serve customers from [primary location] and the greater [city] area, including [list 5-8 areas]."
Tone: helpful, not salesy. No keyword stuffing. One paragraph.
```

## Meta Description
```
Write a meta description (max 160 chars) for a [service business] website in [location].
Formula: Problem + Solution + CTA.
Include primary keyword + location.
```
