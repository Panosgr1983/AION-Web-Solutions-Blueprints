## Philosophy

1. **Knowledge → Asset**: Every client project produces reusable knowledge.
2. **Repeatable Systems**: Not one-off sites — scalable blueprints.
3. **Quality over Quantity**: Fewer pages, better quality, stronger brand.
4. **Data-Driven**: GA4 + Search Console + Clarity on every project.
5. **Score Before Commit**: Project scoring framework prevents bad-fit clients.
6. **Archive Everything**: Every project archived with quality score + lessons learned.
