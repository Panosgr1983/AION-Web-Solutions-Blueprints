# Post-Launch SOP — Service Business v1

## Overview
Standard operating procedure for the 30 days following a website launch.

---

## Day 1: Launch

### Final Checks
- [ ] DNS propagated (if custom domain)
- [ ] SSL active
- [ ] Live test all pages on mobile + desktop
- [ ] Test form submission
- [ ] Test all anchor links
- [ ] Test dark/light mode toggle
- [ ] Test language switch
- [ ] No console errors

### Analytics Activation
- [ ] Verify GA4 is receiving events (Realtime report)
- [ ] Verify Clarity is recording sessions
- [ ] Submit sitemap to Search Console
- [ ] Request manual indexing for homepage
- [ ] Verify OG preview via Facebook Sharing Debugger

### Client Notification
- [ ] Send launch confirmation email
- [ ] Provide admin access links (GA4, Search Console, Clarity)
- [ ] Share 3 action items for client:
  1. Request 5 reviews from recent customers
  2. Share website on Facebook page
  3. Add website link to GBP

---

## Week 1: Monitoring

### Performance
- [ ] Check Lighthouse from production (not local)
- [ ] Monitor Core Web Vitals in Search Console
- [ ] Check page load speed in Clarity recordings

### User Behavior (Clarity)
- [ ] Review heatmaps: where are users clicking?
- [ ] Review scroll depth: do users reach Contact section?
- [ ] Check rage clicks / dead clicks (UX issues)
- [ ] Note any surprising behavior patterns

### Traffic (GA4)
- [ ] Check traffic sources: organic, direct, social, referral
- [ ] Check top landing pages
- [ ] Check CTA button clicks (if event tracking set up)
- [ ] Check phone number clicks (if tracked)
- [ ] Note baseline numbers for comparison

### Search Console
- [ ] Check for crawl errors
- [ ] Check indexing status
- [ ] Review first queries and impressions

### Client Follow-up
- [ ] Check if client posted on social media
- [ ] Check if new reviews have come in
- [ ] Quick check-in: "Everything working well?"

---

## Month 1: Analysis

### Performance Review
- [ ] Compare Week 1 vs Week 4 traffic (GA4)
- [ ] Check organic impressions growth (Search Console)
- [ ] Calculate average position for target keywords
- [ ] Review Clarity recordings for improvement opportunities

### Conversion Optimization
- [ ] Which CTAs receive most clicks?
- [ ] Do users reach the Contact form?
- [ ] Is the phone number clicked more on mobile or desktop?
- [ ] Any pages with high exit rate?

### Content Updates
- [ ] Add any new FAQ questions from real customer inquiries
- [ ] Update service descriptions if needed
- [ ] Add new reviews to testimonials section

### Client Report
- [ ] One-page summary (no jargon)
- [ ] Traffic trend (up/down)
- [ ] Keyword rankings
- [ ] Review count change
- [ ] Recommendations for next month

---

## Month 3: Optimization

### Deep Review
- [ ] Compare vs baseline (Month 1 data)
- [ ] Identify top 3 performing pages
- [ ] Identify top 3 underperforming pages
- [ ] Review competitor changes
- [ ] Update GBP posts

### Recommendations
- [ ] Blog/content ideas based on Search Console queries
- [ ] GBP optimization suggestions
- [ ] Paid ads potential (Google LSA, Facebook)
- [ ] Recurring maintenance proposal (if applicable)

### Client Meeting
- [ ] Present 3-month results
- [ ] Discuss next steps
- [ ] Renew or upsell maintenance/retainer
