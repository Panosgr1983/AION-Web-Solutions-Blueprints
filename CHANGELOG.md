# Changelog — AION Web Solutions Framework

## v1.0 (June 2026) — Initial Release

**Status:** Production Tested ✅
**Reference Project:** NextStage Service Center

### Added
- **Blueprint**: Service Business v1 — full page architecture, conversion rules, trust framework, SEO framework, tech stack
- **Starter Template**: Production-ready Vite + React 18 + TypeScript + Tailwind 4 + Framer Motion
  - 10 baseline sections (Hero → Contact)
  - Dark/light mode with localStorage persistence
  - EL/EN bilingual with flag-only toggle
  - SEO schema (LocalBusiness + FAQPage + areaServed)
  - Performance: WebP, lazy loading, CLS prevention, fetchpriority
- **Sales System**: 5 documents — Discovery Call → Proposal → Pricing → Scope → Onboarding
- **Client Intake**: 3-level questionnaire (Business / Marketing / Technical)
- **Checklists**: project-scoring, gbp-optimization, launch-QA
- **Documentation**: post-launch SOP, quality score, client asset vault, metrics framework
- **Quality Framework**: AION Quality Score — minimum Gold (85/100) before delivery
- **Project Archive**: NextStage with Lessons Learned + Metrics template
- **Playbooks Framework**: Directory structure + methodology template
- **Prompts**: AI copywriting prompts for hero, FAQ, service area, meta

### Principles
1. Knowledge → Asset
2. Repeatable systems over one-off sites
3. Quality over quantity — fewer pages, better quality, stronger brand
4. Data-driven — GA4 + Search Console + Clarity mandatory
5. Score before commit — project scoring prevents bad-fit clients
